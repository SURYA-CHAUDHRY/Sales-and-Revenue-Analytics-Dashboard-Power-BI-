# Sales & Revenue Performance Dashboard (Power BI)

This project analyzes sales performance across different product categories, regions, and time periods. The dataset was prepared in Excel and visualized using Power BI to gain meaningful insights into revenue trends and business performance.

## Key Features
- Interactive **Power BI dashboard** with dynamic filters (slicers) for **Category**, **Region**, and **Month**.
- Custom **DAX measures** including:
  - Total Revenue
  - Total Quantity Sold
  - Average Revenue per Unit
  - Month-wise Sales Trend
- Visualizations include:
  - **Bar Charts** for Monthly and Regional Sales
  - **Pie Chart** for Category-wise Contribution
  - **KPI Cards** for revenue and quantity insights

## Objectives
- Determine top-performing product categories.
- Compare revenue distribution across regions.
- Identify monthly growth patterns.
- Improve decision-making with interactive insights.

## Tools Used
| Tool | Purpose |
|------|---------|
| **Microsoft Excel** | Data cleaning & preprocessing |
| **Power BI** | Data modeling & dashboard creation |
| **DAX** | Calculated measures and KPIs |

## Project Outcome
The dashboard provides clear, actionable insights and allows stakeholders to explore sales trends interactively. It can assist in strategic planning, product focus decisions, and performance review.
